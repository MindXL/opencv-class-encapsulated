# opencv-class-encapsulated
Encapsulate opencv with Object Oriented methods in C++.

While learning opencv, I found it was written in Produce Oriented methods. I decided to learn it by encapsulating it using Object Oriented methods.

# VS 项目设置样例

VC++目录-包含目录：E:\opencv\build\include
VC++目录-库目录：E:\opencv\build\x64\vc15\lib
连接器-输入-附加依赖项：E:\opencv\build\x64\vc15\lib\*d.lib
